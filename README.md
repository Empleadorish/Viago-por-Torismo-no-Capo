# Viagos por Turismo no Capo

Welcome to **Viagos por Turismo no Capo**, a responsive, interactive, and tourist-friendly website for booking guided tours in South Africa. This project is built using **React**, **Vite**, and **Tailwind CSS**.

---

## Features

- Hero section with call-to-action
- Popular tours cards with details
- About section
- Tour gallery
- **Real booking form** powered by Formspree
- Contact form
- Mobile-friendly navigation
- Framer Motion animations for smooth transitions
- Responsive design with Tailwind CSS
- GitHub Pages ready deployment

---

## Technologies Used

- React 18
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React icons
- Formspree for booking form
- GitHub Pages for deployment

---

## Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/Empleadorish/Viago-por-Torismo-no-Capo.git
cd Viago-por-Torismo-no-Capo
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run development server

```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) in your browser.

### 4. Build for production

```bash
npm run build
```

### 5. Deploy to GitHub Pages

```bash
npm run deploy
```
